## 1. 快速开始

### 1.1 登录系统

打开浏览器，访问系统地址（默认 `http://localhost:3000`，局域网用 `http://服务器IP:3000`）。

| 项目 | 默认值 |
|------|--------|
| 用户名 | `admin` |
| 密码   | `admin123` |

> **首次登录后请立即修改默认密码！** 点击左下角用户名 → 修改密码。

### 1.2 推荐浏览器

Chrome、Edge、Firefox 等现代浏览器均可，推荐 **Chrome**。

### 1.3 建议的上手顺序

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 创建用户 | 给团队成员分配账号和角色 |
| 2 | 建立仓库 | 至少需要一个仓库 |
| 3 | 创建物料分类 | 如：电子元件、结构件、包装材料 |
| 4 | 录入物料 | 逐条或批量导入 |
| 5 | 入库操作 | 在仓库管理中登记初始库存 |
| 6 | 日常使用 | 出入库、发货、查看报表 |

## 2. 系统概览

### 2.1 功能模块

```
┌──────────────────────────────────────────────┐
│                 OvO System                  │
├──────────┬──────────┬──────────┬─────────────┤
│ 业务管理 │   生产   │   数据   │    系统     │
├──────────┼──────────┼──────────┼─────────────┤
│ 物料管理 │ SOP管理  │ 数据统计 │ 用户管理    │
│ 仓库管理 │ BOM管理  │ 报表中心 │ 权限控制    │
│ 发货管理 │ 生产管理 │          │ 操作日志    │
└──────────┴──────────┴──────────┴─────────────┘
```

### 2.2 界面布局

- **左侧导航栏**：点击切换功能模块，底部可以折叠/展开
- **顶部标题栏**：显示当前页面名称 + 全局物料搜索框（支持拼音）
- **主内容区**：表格、卡片、图表等内容展示
- **弹窗（Modal）**：新增/编辑/详情等操作在弹窗中完成

### 2.3 全局搜索

顶部搜索框可快速搜索物料，支持：
- 物料名称（如"连接器"）
- 拼音全拼（如"lianjieqi"）
- 拼音首字母（如"ljq"）
- 物料编码（如"MAT-20260318-001"）

按 **Enter** 键跳转到物料管理页面显示搜索结果。

## 3. 仪表盘

登录后首先看到的页面，一目了然掌握系统状态。

### 3.1 统计卡片

| 卡片 | 说明 | 点击跳转 |
|------|------|----------|
| 物料总数 | 系统中已录入的物料种类数 | 物料管理 |
| 仓库数量 | 已创建的仓库数 | 仓库管理 |
| 待发货订单 | 状态为"待发货"的发货单数 | 发货管理 |
| 低库存预警 | 库存低于最小库存的物料数，红色标识 | — |

### 3.2 低库存预警

列出所有库存不足的物料，显示：
- 物料名称和编码
- 当前库存 / 最小库存
- 缺口数量（红色字体）

### 3.3 今日概览

显示当天的入库总量、出库总量、库存总价值。

### 3.4 最近操作

显示系统中最近的操作记录（谁在什么时间做了什么），方便团队了解动态。

## 4. 物料管理

> 路径：业务管理 → 物料管理

### 4.1 物料列表

以表格展示所有物料信息：

| 列 | 说明 |
|----|------|
| 编码 | 系统自动生成，格式 `MAT-YYYYMMDD-NNN` |
| 名称 | 物料名称 |
| 分类 | 所属分类（如电子元件、结构件） |
| 规格 | 规格型号 |
| 品牌 | 品牌名称 |
| 单位 | 计量单位（个、块、片等） |
| 库存 | 当前总库存数量 |
| 成本价 | 单件成本价格 |
| 状态 | 正常/低库存（低库存用红色标记） |

**搜索**：顶部搜索框支持名称、拼音、编码搜索，实时筛选。

**分类筛选**：点击分类下拉框可按分类查看。

### 4.2 新建物料

点击 **「+ 新建物料」** 按钮，填写以下信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | ✅ | 物料名称 |
| 分类 | - | 下拉选择已有分类 |
| 单位 | ✅ | 默认"个"，可改为块、片、米等 |
| 规格 | - | 型号规格 |
| 品牌 | - | 品牌名称 |
| 成本价 | - | 单件采购/制造成本 |
| 售价 | - | 单件销售价格 |
| 最小库存 | - | 低于此值触发库存预警 |
| 供应商 | - | 供应商名称 |
| 备注 | - | 其他信息 |

> 编码无需手动输入，系统按日期自动生成。

### 4.3 编辑/删除物料

- **编辑**：点击物料行右侧的编辑图标
- **删除**：点击删除图标（会弹出确认提示）

> 删除为软删除，不会影响历史出入库记录。

### 4.4 管理分类

点击 **「分类」** 按钮打开分类管理弹窗：
- 输入分类名称 → 点击"添加"创建新分类
- 点击分类右侧的删除按钮可删除空分类（有物料引用的分类不可删除）

### 4.5 批量导入

点击 **「导入」** 按钮，详见 [第13章 数据导入导出](#13-数据导入导出)。

## 5. 仓库管理

> 路径：业务管理 → 仓库管理

### 5.1 仓库列表

每个仓库显示为一张卡片，包含：
- 仓库名称和地址
- 物料品种数量
- 库存总量
- 联系人信息
- 操作按钮（编辑、删除）

### 5.2 新建仓库

点击 **「+ 新建仓库」**，填写：

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | ✅ | 仓库名称 |
| 地址 | - | 仓库地址 |
| 联系人 | - | 负责人姓名 |
| 联系电话 | - | 负责人电话 |
| 备注 | - | 其他信息 |

### 5.3 出入库操作

点击仓库卡片上的 **「库存管理」** 按钮，进入出入库操作：

#### 入库
1. 选择操作类型 → **入库**
2. 选择物料（支持搜索）
3. 输入数量
4. 填写备注（可选，如采购单号）
5. 点击 **「确认入库」**

#### 出库
1. 选择操作类型 → **出库**
2. 选择物料
3. 输入数量（不能超过当前库存）
4. 填写备注
5. 点击 **「确认出库」**

> 每次出入库都会自动记录流水，可在报表中心查看。

## 6. 发货管理

> 路径：业务管理 → 发货管理

### 6.1 发货单列表

以表格展示所有发货单：

| 列 | 说明 |
|----|------|
| 发货单号 | 自动生成，格式 `SHP-YYYYMMDD-NNN` |
| 客户 | 客户名称 |
| 物料数 | 包含的物料种类数 |
| 总金额 | 订单总金额 |
| 状态 | 待发货/已发货/已送达/已取消 |
| 创建时间 | 创建日期 |

**状态说明：**

| 状态 | 颜色 | 含义 |
|------|------|------|
| 待发货 | 黄色 | 已创建，等待发货 |
| 已发货 | 蓝色 | 已出库发货 |
| 已送达 | 绿色 | 客户已签收 |
| 已取消 | 红色 | 订单取消，库存已回退 |

### 6.2 新建发货单

点击 **「+ 新建发货单」**：

1. **填写客户信息**
   - 客户名称
   - 联系方式
   - 发货地址
   - 选择出库仓库

2. **添加发货物料**
   - 点击 **「+ 添加物料」**
   - 选择物料（下拉搜索）
   - 输入发货数量和单价
   - 可添加多个物料

3. 检查无误后点击 **「创建」**

> 创建发货单时系统会检查库存是否充足。

### 6.3 发货单状态流转

点击发货单的状态下拉框可更新状态：

```
待发货 → 已发货 → 已送达
   ↓
 已取消（库存自动回退）
```

## 7. SOP管理

> 路径：生产 → SOP管理

SOP（标准操作流程）用于规范生产步骤和物料用量。

### 7.1 SOP列表

卡片式展示，每张卡片显示：
- SOP标题和版本号
- 所属分类
- 步骤数量
- 关联物料数
- 创建人

### 7.2 新建SOP

点击 **「+ 新建SOP」**：

1. **基本信息**
   - 标题（必填）
   - 版本号（默认 1.0）
   - 分类（如：装配、测试、包装）
   - 描述

2. **添加步骤**
   - 点击 **「+ 添加步骤」**
   - 填写步骤标题、详细说明、预计用时
   - 可添加多个步骤，步骤自动编号

3. **关联物料**
   - 点击 **「+ 添加物料」**
   - 选择所需物料和每单位用量
   - 可选择关联到具体步骤

### 7.3 查看/复制SOP

- 点击 SOP 卡片查看详情（步骤流程 + 物料清单）
- 点击 **「复制」** 基于现有 SOP 创建副本

## 8. BOM管理

> 路径：生产 → BOM管理

BOM（物料清单）定义一个产品/组件由哪些物料或子组件组成，支持多级嵌套。

### 8.1 BOM列表

卡片式展示，每张卡片显示：
- BOM名称和编码
- 版本号和状态（生效/草稿/归档）
- 产出物料名称
- 物料项数和子件数
- 创建人

### 8.2 新建BOM

点击 **「+ 新建BOM」**：

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | ✅ | BOM名称，如"信号增强器组件" |
| 状态 | - | 生效/草稿/归档 |
| 产出物料 | - | 此BOM生产出的成品物料 |
| 产出数量 | - | 一次生产的成品数量 |
| 分类 | - | BOM分类，如"电子组件" |
| 描述 | - | 详细说明 |

**添加物料清单：**
- 点击 **「添加物料」**：添加具体物料项，填写用量、位号、损耗率
- 点击 **「添加子BOM」**：引用另一个BOM作为子组件（实现多级嵌套）

**字段说明：**

| 字段 | 说明 | 示例 |
|------|------|------|
| 用量 | 每个成品需要的数量 | 2 |
| 位号 | 物料在产品中的位置标记 | J1,J2 |
| 损耗% | 预计损耗比例，影响实际用量计算 | 2（表示2%） |

### 8.3 BOM详情

点击BOM卡片查看详情，展示两个核心表格：

**① 物料清单（多级展开）**

树形结构展示BOM的所有层级：
- 子BOM用紫色背景和分支图标标识
- 子BOM下的物料自动缩进展示
- 每行显示：物料名称、规格、位号、用量、损耗率、实际用量、行成本
- 底部显示 **合计成本**

**② 原材料需求汇总**

将所有层级的叶子物料合并汇总：
- 相同物料的用量自动累加
- 显示每种原材料的总用量、单价、总成本

### 8.4 版本管理

每次编辑BOM时系统会自动：
1. 保存当前版本的快照
2. 自动递增版本号（1.0 → 1.1 → 1.2 ...）

点击详情中的 **「版本历史」** 按钮可以：
- 查看所有历史版本及变更说明
- 点击 **「恢复此版本」** 回滚到任一历史版本（当前版本会自动备份）

### 8.5 复制BOM

在详情弹窗中点击 **「复制」**，基于当前BOM创建一个副本，方便创建相似产品的BOM。

## 9. 生产管理

> 路径：生产 → 生产管理

基于SOP创建生产工单，追踪生产进度。

### 9.1 工单列表

| 列 | 说明 |
|----|------|
| 工单号 | 自动生成，格式 `PO-YYYYMMDD-NNN` |
| SOP | 关联的标准操作流程 |
| 计划数量 | 计划生产数量 |
| 完成数量 | 已完成数量 |
| 进度 | 进度条显示完成百分比 |
| 状态 | 计划中/进行中/已完成/已取消 |

**状态筛选**：顶部下拉框可按状态筛选工单。

### 9.2 创建工单

点击 **「+ 创建工单」**：

| 字段 | 必填 | 说明 |
|------|------|------|
| SOP | ✅ | 选择要执行的SOP |
| 仓库 | ✅ | 物料领取的仓库 |
| 计划数量 | ✅ | 计划生产数量 |
| 产出物料 | - | 生产完成后入库的物料 |
| 备注 | - | 生产说明 |

### 9.3 工单状态流转

点击工单可查看详情并更新状态：

```
计划中 → 开始生产 → 完成生产
  ↓                    ↓
取消工单           （自动扣减物料库存）
                   （产出物料自动入库）
```

详情页会显示所需物料清单及当前库存情况，方便确认备料状态。

## 10. 数据统计

> 路径：数据 → 数据统计

### 10.1 时间范围

顶部可选择统计时间范围：
- 快捷选择：最近7天 / 30天 / 90天
- 自定义日期范围

### 10.2 统计卡片

| 指标 | 说明 |
|------|------|
| 期间入库 | 选定时间范围内的入库总量 |
| 期间出库 | 选定时间范围内的出库总量 |
| 入库金额 | 入库物料的总金额 |
| 出库金额 | 出库物料的总金额 |

### 10.3 图表

| 图表 | 类型 | 展示内容 |
|------|------|----------|
| 出入库趋势 | 折线图 | 按日期展示入库/出库数量变化趋势 |
| 分类库存价值 | 饼图 | 各分类的库存价值占比 |
| 仓库库存对比 | 柱状图 | 各仓库的库存数量对比 |

### 10.4 排行榜

- **出库TOP10**：出库量最大的10种物料
- **库存价值TOP10**：库存金额最高的10种物料
- **入库TOP10**：入库量最大的10种物料

## 11. 报表中心

提供三个报表标签页，均支持导出。

- 数量、金额
- 单据号、操作人

### 11.3 发货统计

- 发货趋势图（按时间）
- 热销物料TOP10
- 热门客户/目的地TOP10

### 12.2 权限对照表

| 功能模块 | 管理员 | 编辑者 | 查看者 |
|----------|--------|--------|--------|
| 物料管理 | 增删改查 | 增改查 | 查 |
| 仓库管理 | 增删改查 | 增改查 | 查 |
| 发货管理 | 增删改查 | 增改查 | 查 |
| SOP管理 | 增删改查 | 增改查 | 查 |
| BOM管理 | 增删改查 | 增改查 | 查 |
| 生产管理 | 增删改查 | 增改查 | 查 |
| 数据统计 | 查 | 查 | 查 |
| 报表中心 | 查 | 查 | 查 |
| 用户管理 | 增删改查 | ✗ | ✗ |

> 没有权限的操作按钮在界面上自动隐藏，后端也会拦截。

### 12.3 新建用户

点击 **「+ 新建用户」**：

| 字段 | 必填 | 说明 |
|------|------|------|
| 用户名 | ✅ | 登录用名，不可重复 |
| 密码 | ✅ | 至少6位 |
| 显示名称 | ✅ | 界面上展示的名字 |
| 角色 | ✅ | 管理员/编辑者/查看者 |
| 启用状态 | - | 可禁用用户而不删除 |

### 12.4 修改密码

每个用户都可以修改自己的密码：
1. 点击左下角的 **「修改密码」** 按钮
2. 输入当前密码
3. 输入新密码（至少6位）
4. 确认新密码
5. 点击确定

## 13. 数据导入导出

### 13.1 导出

几乎所有列表页面都支持导出，点击 **「导出 ▼」** 下拉按钮，选择格式：

| 格式 | 说明 | 适合场景 |
|------|------|----------|
| Excel (.xlsx) | 带样式的Excel表格，含表头格式、交替行色 | 打印、分享给客户 |
| CSV (.csv) | 逗号分隔文本，UTF-8编码 | 其他系统导入、数据处理 |
| JSON (.json) | 结构化数据，中文键名 | 程序对接、数据备份 |

**支持导出的页面：**
物料管理、仓库管理（库存）、发货管理、SOP管理、BOM管理、生产管理、报表中心

### 13.2 导入

目前支持 **物料批量导入**。

#### 操作步骤：

1. 进入物料管理页面，点击 **「导入」** 按钮
2. 先下载模板（支持 XLSX 和 CSV 格式）
3. 按模板格式填写物料数据
4. 将文件拖拽到上传区域，或点击选择文件
5. 系统自动校验并显示导入结果

#### 模板字段说明：

| 列名 | 必填 | 说明 |
|------|------|------|
| 名称 | ✅ | 物料名称 |
| 分类 | - | 分类名称（不存在会自动创建） |
| 单位 | - | 默认"个" |
| 规格 | - | 型号规格 |
| 品牌 | - | 品牌名称 |
| 成本价 | - | 数字 |
| 售价 | - | 数字 |
| 最小库存 | - | 数字 |
| 供应商 | - | 供应商名称 |
| 备注 | - | 其他信息 |

#### 导入规则：
- 编码由系统自动生成，不需要填写
- 如果名称+规格完全相同的物料已存在，该行会被跳过
- 分类名不存在时会自动创建该分类
- 导入结果会显示：成功数、跳过数、失败数及失败原因

## 14. 常见问题

### Q: 忘记密码怎么办？
联系管理员（admin角色），在用户管理中重置密码。

### Q: 物料编码可以自定义吗？
不可以。编码由系统按 `MAT-日期-序号` 格式自动生成，保证唯一性和一致性。

### Q: 删除物料/仓库会影响历史记录吗？
不会。删除是"软删除"，历史出入库记录和发货记录不受影响。

### Q: 出库时提示库存不足怎么办？
说明该物料在对应仓库的库存不够，需要先入库补货或从其他仓库调拨。

### Q: 发货单取消后库存会自动恢复吗？
会。取消已确认的发货单时，已扣减的库存会自动回退。

### Q: BOM的损耗率怎么理解？
损耗率表示生产过程中的材料损耗比例。例如设置 2%，用量为100，实际用量计算为 100 × 1.02 = 102。

### Q: 多个人可以同时使用系统吗？
可以。系统支持多用户同时在线操作，每人用自己的账号登录。

### Q: 数据存储在哪里？
所有数据存储在项目目录下的 `data/inventory.db` 文件中（SQLite数据库）。建议定期运行 `deploy\backup.bat` 备份。

### Q: 如何从其他电脑访问？
在服务器上运行 `deploy\open-firewall.bat`（管理员权限），之后同一局域网内的电脑用 `http://服务器IP:3000` 访问即可。

### Q: 系统支持手机访问吗？
系统界面基于 Web 技术，手机浏览器可以打开访问，但未针对小屏幕做专门适配，建议在电脑上使用。
