# OvO Inventory System 用户手册

适用对象：仓库管理员、生产主管、业务跟单、管理人员  
目标：说明日常怎么录单、查单、查库存、查报表。

## 1. 快速开始

访问地址：
- 本机：`http://localhost:3000`
- 局域网：`http://服务器IP:3000`

默认账号：
- 用户名：`admin`
- 密码：`admin123`

首页主要看四类信息：
- 风险层：低库存、保供套数、高风险供应
- 动作层：高频执行入口
- 核算层：库存价值、物料数量
- 管理层：最近动态、趋势图表

## 2. 导航怎么理解

- `物料`：物料主档、分类、阈值、供应商、替代料
- `收`：收货入库、完工入库、退料回库
- `发`：发料出库、销售发货、生产领料
- `转`：仓间调拨、车间转仓
- `盘`：盘点调整、库存校正
- `生产`：BOM、SOP、生产工单
- `报表`：正式报表、动作流水、一致性治理
- `系统`：仓库资料、用户与权限

一句话：
- 录库存动作，用 `收 / 发 / 转 / 盘`
- 定义生产规则，用 `生产`
- 查数据和问题，用 `报表`

## 3. 物料怎么用

路径：`物料`

### 3.1 新建物料

点击 `新建物料`，优先填这些字段：
- 名称
- 分类
- 规格
- 单位
- 品牌
- 成本价
- 安全库存 / 最低库存 / 补货点
- 目标保供套数
- 供应商信息

建议顺序：
1. 先录名称、分类、单位、规格
2. 再补成本、阈值、供应商

### 3.2 查物料

支持搜索：
- 名称
- 编码
- 拼音
- 规格

不要靠滚动列表找，直接用搜索框。

### 3.3 看物料详情

两种方式打开：
- 直接点击整行物料
- 点击右侧 `详情`

详情里能看：
- 基本信息
- 当前库存
- 最近流水
- 供应商与报价记录
- 被哪些 BOM / SOP / 工单引用
- 替代料
- 供应风险

### 3.4 导入

点击 `导入`，当前支持：
- 模板导入
- 库存状况表
- 供应商价格本

统一流程：
1. 选择导入类型
2. 上传文件
3. 先预览
4. 确认无误后提交

## 4. 收 / 发 / 转 / 盘 怎么操作

这四个模块是仓库和车间最常用入口。

### 4.1 收货入库

路径：`收 > 收货入库`  
适用：采购到货、完工入库、退料回库

步骤：
1. 选择仓库
2. 选择物料
3. 填数量
4. 填往来方或备注
5. 提交

两种提交方式：
- `提交单据`：只生成单据，不立即记账
- `一键提交并执行记账`：立即生效，适合日常使用

### 4.2 发料出库

路径：`发 > 发料出库`  
适用：手工出库、临时发料、补充出库

步骤：
1. 选择仓库
2. 选择物料
3. 填数量
4. 填用途、往来方或备注
5. 提交

注意：
- 出库数量不能超过可用库存
- 想立即生效，用 `一键提交并执行记账`

### 4.3 仓间调拨

路径：`转 > 仓间调拨`

步骤：
1. 选源仓
2. 选目标仓
3. 选物料
4. 填调拨数量
5. 填参考号或备注
6. 提交

### 4.4 盘点调整

路径：`盘 > 盘点调整`

步骤：
1. 选仓库
2. 选物料
3. 查看账面库存
4. 输入实盘数量
5. 系统自动计算差值
6. 提交

注意：盘点调整会直接改变库存余额，备注建议写清原因。

## 5. 单据怎么管理、怎么查询

### 5.1 单据状态

- `draft`：草稿
- `submitted`：已提交，待执行
- `executed`：已执行，库存动作已发生
- `posted`：已记账，进入正式台账
- `voided`：已作废

### 5.2 在哪里查单据

`收 / 发 / 转 / 盘` 页面都能查本模块单据，通常有两种视图：
- `流水视图`：看每条库存动作
- `单据视图`：按单号看整张单

常用筛选：
- 日期范围
- 仓库
- 物料
- 单号
- 往来方
- 来源标签

### 5.3 怎么判断单据是否真正生效

不要只看“提交成功”，要看：
- 单据状态是否到 `executed / posted`
- 动作流水里是否查得到
- 库存是否真的变化

如果只是 `submitted`，说明单据已生成，但不一定已记账。

### 5.4 独立单据查询页

当前这些单据已经提供独立查询页，不需要只在统一执行页或工单详情里找：

- `收 > 采购收货单`
- `盘 > 库存调整单`
- `生产 > 生产领料单`
- `生产 > 完工入库单`
- `生产 > 退料回库单`
- `生产 > 生产异常单`

这些页面适合做：
- 按单号、物料、仓库、日期查单
- 按状态筛选
- 打开详情追溯来源工单
- 从异常或治理场景回查对应单据

## 6. 销售发货怎么操作

路径：`发 > 销售发货`

### 6.1 新建发货单

步骤：
1. 点击 `新建销售发货`
2. 填客户信息
3. 在明细区选物料
4. 填数量
5. 核对单价和金额
6. 保存

说明：新建时会默认生成一行空明细，不需要先点“添加物料”。

### 6.2 发货状态

- `pending`：待处理
- `confirmed`：已确认待发
- `shipped`：已发出
- `cancelled`：已取消

发出后才真正扣库存；取消会走正式回冲。

### 6.3 怎么查发货单

可按以下条件查询：
- 客户
- 状态
- 单号
- 日期范围

详情里可以看：
- 发货单基本信息
- 物料明细
- 对应库存单据
- 库存单据状态

## 7. BOM / SOP / 生产工单

### 7.1 BOM

路径：`生产 > BOM结构`

用途：定义一个成品或半成品由哪些物料组成。

新建 BOM 要填：
- BOM 名称
- 产出物料
- 产出数量
- 明细物料
- 用量
- 位置
- 损耗率
- 是否允许替代

支持：
- 批量粘贴物料
- 搜索式选料
- 缺料检查
- 成本分析

### 7.2 SOP

路径：`生产 > SOP工艺`

用途：定义生产步骤、工艺顺序、步骤物料。

### 7.3 生产工单

路径：`生产 > 生产工单`

常见流程：
1. 创建工单
2. 检查是否齐套
3. 开始生产
4. 领料
5. 完工入库
6. 退料
7. 如有问题记录异常

工单详情重点看：
- 领料需求与库存检查
- 高风险供应料提示
- BOM / SOP 快照
- 关联领料单、完工单、退料单
- 生产异常单

### 7.4 生产单据独立查询

如果你不是要改工单，而是要查生产过程中的单据，优先走这些入口：

- `生产 > 生产领料单`
- `生产 > 完工入库单`
- `生产 > 退料回库单`
- `生产 > 生产异常单`

适用场景：
- 仓库查某张领料单是否已记账
- 生产主管查某张完工单对应哪个工单
- 查退料是否已真正回库
- 查异常单是否已经处理、谁在跟进

### 7.5 生产异常治理怎么用

路径：`生产 > 生产异常单`

常见用法：
1. 按异常类型、状态、工单号筛选
2. 打开详情看异常原因和关联单据
3. 在治理区维护：
   - 治理状态
   - 责任人
   - 处理备注
4. 如异常涉及高风险供应料，可点 `供应治理`

治理状态可理解为：
- `待处理`：问题已登记，尚未处理
- `处理中`：有人跟进中
- `已处理`：现场已完成处理
- `已忽略`：确认不需要继续跟进

## 8. 预警怎么看

### 8.1 低库存预警

表示当前库存低于安全库存或最低库存。  
点进去可看：
- 当前库存
- 阈值
- 缺口

### 8.2 保供套数预警

表示当前库存不能满足目标保供套数。  
适合看：
- 当前最多还能做几套
- 是哪个瓶颈料卡住了生产

### 8.3 高风险供应预警

表示该料有更高供应风险，常见原因：
- 唯一供应商
- 长交期
- 无替代料
- 关键件
- 覆盖天数不足

适合采购、计划、生产主管重点看。

点进去后重点看：
- 风险原因
- 覆盖天数
- 是否唯一供应商
- 是否有替代料
- 建议动作

当前建议动作已经能形成任务：
- `生成采购任务`
- `生成备料任务`
- `生成替代治理任务`

## 9. 报表怎么查

路径：`报表`

### 9.1 分析看板

看趋势与概览，适合管理层快速查看：
- 收 / 发 趋势
- 库存价值
- 生产异常概览
- 供应风险预警

### 9.2 正式报表

常用页面：
- 库存余额
- 动作流水
- 销售发货分析
- 高风险供应治理

### 9.2.1 高风险供应治理

路径：`报表 > 正式报表 > 高风险供应治理`

适合谁用：
- 采购
- 计划
- 生产主管
- 管理人员

当前可做的事：
1. 按风险等级、责任人、任务类型筛选
2. 看系统建议的采购/备料/替代治理动作
3. 直接生成或更新治理任务
4. 维护：
   - 任务类型
   - 治理状态
   - 责任人
   - 处理备注

任务类型包括：
- `采购任务`
- `备料任务`
- `替代治理任务`

### 9.3 一致性治理

路径：
- 左侧导航 `一致性治理`
或
- `报表 > 一致性治理`

主要查：
- 负库存
- 余额 / 流水不一致
- 无流水历史基线

可直接：
- 查看流水
- 导出 Excel
- 按问题类型治理
- 维护责任人、处理状态和备注

当前治理状态包括：
- `待处理`
- `处理中`
- `已处理`
- `已忽略`

### 9.4 动作流水

用途：
- 查所有库存动作
- 按物料、仓库、日期追溯

适合核对：
- 某物料最近怎么变动
- 某单据是否真正影响了库存

### 9.5 命名治理

当前系统已经提供两类命名治理入口：

- `物料 > 命名治理`
- `生产 > BOM结构 > 命名治理`

用途：
- 找出不合规范的名称
- 看命名问题标签
- 看建议规范名
- 再回到物料或 BOM 详情做整改

常见问题标签包括：
- 名称过短
- 含日期
- 含“测试 / 模板 / 最新版”一类噪音词
- 缺少规格、单位、编码或版本信息

## 10. 常见查询场景

### 查某个物料当前库存

1. 打开 `物料`
2. 搜索该物料
3. 打开详情
4. 看库存和最近流水

### 查某个物料被哪些 BOM 用到

1. 打开物料详情
2. 看 `业务引用详情 > BOM`
3. 区分直接引用和间接引用
4. 点 `查看BOM`

### 查某张单据是否已记账

1. 进入对应模块
2. 切到 `单据视图`
3. 搜索单号
4. 看状态是否为 `posted`

### 查问题物料的历史动作

1. 进入 `一致性治理`
2. 找到问题物料
3. 点 `查看流水`
4. 系统会跳到独立 `动作流水` 页面

## 12. 常见问题

### 页面提示成功，为什么库存没变

通常是因为你只是 `提交单据`，还没有执行记账。  
请查看单据状态是否到了 `executed / posted`。

### 为什么有库存还预警

可能原因：
- 低于安全库存
- 低于最低库存
- 保供套数不足
- 属于高风险供应料

### 导入失败怎么办

先确认：
- 导入类型选对没有
- 是否先做了预览
- 错误明细提示是什么

### 查不到某张单据怎么办

先检查：
- 是否在正确模块
- 当前看的是不是 `单据视图`
- 日期范围是否太窄
- 单据是否还只是草稿

## 13. 推荐上手顺序

新用户建议：
1. 先看首页和高频入口
2. 学会在 `物料` 中查详情
3. 学会 `收 / 发 / 转 / 盘`
4. 学会查单据和动作流水
5. 再学 `销售发货`
6. 最后再学 `BOM / SOP / 生产工单`

如果只是仓库日常执行，先学会这 5 项就够：
- 收货入库
- 发料出库
- 仓间调拨
- 盘点调整
- 单据查询
